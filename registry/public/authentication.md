# Authentication

The public registry uses personal access tokens (PATs) to authorize requests.

## Steps to Generate a Token

1. Log in to your account
2. Go to the [PAT section](https://trmregistry.com/pat).
3. Press **Generate new token**
4. Select an expiration date (or no expiration) and the desired scopes
5. Press **Generate token**

> This is your **only chance** to copy the token. Make sure to store it somewhere safe.
